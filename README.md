# odin-recipes

This is the **first** true TOP assignment. The goal is to create a simple recipes website

The goal of this project is to:
* Gain familiarity with the Git Workflow (add --> commit --> push)
* Learn how to create a website using HTML
* Understand how to link together webpages to create a convenient user experience
* Learn how to properly credit free images when creating a website
